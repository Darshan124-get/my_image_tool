# Assets Directory

This directory should contain:

## og-image.png
- Open Graph image for social media sharing
- Recommended size: 1200×630 pixels
- Should include MyImageTool branding and a preview of the tools
- Format: PNG

## favicon.ico
- Browser favicon
- Recommended sizes: 16×16, 32×32, 48×48 pixels
- Should include MyImageTool logo/icon
- Format: ICO (or PNG for modern browsers)

## Generating Placeholders

You can use online tools or create these images:
- **og-image.png**: Use a design tool (Figma, Canva, etc.) to create a 1200×630px image with MyImageTool branding
- **favicon.ico**: Use a favicon generator (favicon.io, realfavicongenerator.net) or create a simple icon

For quick testing, you can use any placeholder image service or create simple colored images.
